# 《Spring Boot 编程思想》

本书全名为《Spring Boot 编程思想》，是以 Spring Boot 2.0 为讨论的主线，讨论的范围将涵盖 Spring Boot 1.x 的所有版本，以及所关联的 Spring Framework 版本，致力于：

- 场景分析：掌握技术选型
- 系统学习：拒绝浅尝辄止
- 重视规范：了解发展趋势
- 源码解读：理解设计思想
- 实战演练：巩固学习成果
